# 🎼 Classical Music Hitster

Egy Apple Music playlistből generált Hitster webalkalmazás.

## Funkciók

- 🔎 Keresés kártyaszám alapján
- 🎼 Mű címe
- 👤 Zeneszerző
- 🎻 Előadó
- 📅 Megjelenési év
- 🍎 Megnyitás Apple Musicban

## Használat

1. Írd be a kártya sorszámát.
2. Kattints a **Keresés** gombra.
3. Nyomd meg a **Megnyitás Apple Musicban** gombot.

## Fejlesztés

```bash
npm install
node builder/build.js
```

## Technológia

- JavaScript
- Playwright
- Apple Music Web
- GitHub Pages
- Progressive Web App (PWA)
